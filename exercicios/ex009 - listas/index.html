Tipo de Listas

Lista Ordenada

É uma lista que a ordem dos elementos importa. Usamos "ol" e dentro de ol usamos "li" em cada item listado.

EX: Um processo que requer passos especificos pra acontecer.

  1. Acordar
  2. Levantar
  3. Tomar Banho
  4. Tomar Café
  5. Escovar os Dentes
  6. Se arrumar
  7. Ir trabalhar
  8. Voltar pra casa
  9. Assistir Curso em Video
  10. Tomar Banho
  11. Dormir

Lista não Ordenada

É uma lista que a ordem dos elementos não importa. Usamos "ul" e dentro de ul usamos "li" em cada elemento da lista.

EX:Uma Lista de Compra

  • Abacate
  • Feijão
  • Alface
  • Creme de leite
  • Biscoito

Lista dentro de lista

Vamos usar as duas listas anteriores "ol", "ul" e "li". Este padrão geralmente usado para fazer MENUS de sites.
