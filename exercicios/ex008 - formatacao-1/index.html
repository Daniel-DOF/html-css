Estilos de Formatação

Negrito/Destaque

Esta frase foi utilizado negrito utilizando b(não semântico)

Neste caso foi utilizado para destacar o comando STRONG(semântico)

Itálico / Ênfase

Utlizaremos i pra deixar uma fra sem itálico(não semântico)

Agora uzaremos o em para dar ênfase(semântico)

Texto Marcado

Para deixar o texto marcado utiliza-se a tag mark

Sublinhado/Texto Inserido

Utiliza-se u para sublinhar o texto(não semântica)

Pode-se utilizar ins para deixar o texto sublinhado/ou dizer que foi inserido depois

Texto Grande e Texto Pequeno

usa-se big pa almentar um pouco o texto e small para reduzir um pouco o texto.

Texto Sobrescrito

Usa-se a tag sup para deixar o texto sobrescrito ex: a20+b=0

Texto Subescrito

Usa-se a tag sub para deixar o texto subescrito ex: H2O
