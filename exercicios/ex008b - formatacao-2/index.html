Outras Formatções

Código Fonte

O comando "code" é usado para deixar as letras de um texto com o mesmo espaçamento, dando um melhor entendimento do texto lido em alguns casos. ex: www.google.com

Pré-Formatação

O comando "pre" é usado para que o texto digitado seja mostrado com todas as tabulações e quebras de página. Também o comando "code" pode ser usado dentro do comando "pre"

Citações

Citações Completas

Usa-se a tag "blockquote" para este tipo de citações, onde a mesma aparece com tabulação dando destaque ao autor mencionado.

Abreviações

Tag "abbr" para mostrar o significado da abrevição usada. Ex: HTML e CSS.

Texto Invertido

Usa a tag "bdo" para inverter o texto da esqueda para a direita ou da direita para a esquerda.
